  通知機能の動作フロー

  1. ユーザーAがユーザーBの投稿にリアクション（👏、💖、🤣）を付ける
  2. toggleReaction() が実行され、リアクションがデータベースに保存される
  3. createNotification() が呼び出され、ユーザーBに通知が作成される
  4. ヘッダーの通知ベルに未読バッジが表示される（30秒ごとのポーリングで更新）
  5. ベルアイコンをクリックするとドロップダウンで通知一覧が表示される
  6. 通知をクリックすると既読になり、ホームフィードに遷移する
  7. Sidebarの「🔔 通知」リンクから通知ページ(/notifications)にアクセス可能

  4. ユーザーAのヘッダーに未読バッジが表示されることを確認

  6. 通知をクリックして既読になることを確認
